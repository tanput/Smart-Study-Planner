Project Overview

Smart Study Planner is a Java-based console application that helps students organize and manage their study tasks effectively. It allows users to add tasks with deadlines, view all tasks, and mark tasks as completed. The project applies core Java OOP concepts along with file handling to store task data permanently.

Features

Add new study tasks

Set deadlines (YYYY-MM-DD format)

View all saved tasks

Mark tasks as completed

Automatically saves tasks using file handling

Loads previous tasks when the program restarts

Simple menu-driven interface

Project Description

This project helps students plan their academic tasks by letting them store, update, and track deadlines. It uses object-oriented programming principles and serialization to ensure all tasks are saved and loaded automatically. It improves study productivity through simple task management.
